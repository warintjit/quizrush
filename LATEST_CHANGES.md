# สรุปการปรับปรุงล่าสุด — QuizRush

อัปเดตล่าสุด: 17 กรกฎาคม 2026

## ภาพรวม

เพิ่มโหมดฝึกทำข้อสอบแบบเล่นคนเดียว โดยนำชุดข้อสอบที่มีอยู่ในคลังมาใช้ และนำเว็บไซต์เวอร์ชันล่าสุดขึ้นใช้งานบน Netlify Production

## ฟังก์ชันโหมดฝึกคนเดียว

- เพิ่มปุ่ม **ฝึกทำข้อสอบคนเดียว** ในหน้าแรก
- เพิ่มหน้า `/practice` สำหรับตั้งค่าการฝึก
- เลือกชุดข้อสอบจากคลังข้อสอบเดิมได้
- กำหนดชื่อผู้ฝึก สีอวตาร และระยะเวลาฝึก 1–30 นาทีได้
- หากไม่มีชุดข้อสอบ ระบบจะใช้คำถามตัวอย่าง
- เมื่อเริ่มฝึก ระบบจะสร้างห้องส่วนตัวและเริ่มเกมให้อัตโนมัติ
- เมื่อทำครบทุกข้อ ระบบจะวนข้อที่ฝึกน้อยที่สุดให้โดยอัตโนมัติ
- แยกสถานะการเล่นด้วย `mode: practice` เพื่อให้หน้าการเล่นรองรับโหมดฝึก

## ไฟล์หลักที่ปรับปรุง

- `src/pages/Practice.jsx` — หน้าตั้งค่าและเริ่มโหมดฝึก
- `src/App.jsx` — เพิ่ม route `/practice`
- `src/pages/Home.jsx` — เพิ่มทางเข้าโหมดฝึกจากหน้าแรก
- `src/pages/Playing.jsx` — รองรับพฤติกรรมของโหมดฝึก
- `src/index.css` — เพิ่มรูปแบบการแสดงผลสำหรับหน้าโหมดฝึก
- `.gitignore` — ไม่ติดตามไฟล์ local ของ Netlify

## การนำขึ้นใช้งานจริง

- Production URL: <https://pcm-quizrush.netlify.app>
- หน้าโหมดฝึก: <https://pcm-quizrush.netlify.app/practice>
- ตั้งค่า Environment Variables ที่จำเป็นบน Netlify:
  - `VITE_SUPABASE_URL`
  - `VITE_SUPABASE_ANON_KEY`
- แก้ปัญหาหน้าจอดำจากการ build โดยไม่มีค่า Supabase
- ตรวจสอบแล้วว่าหน้าแรกเปิดได้ หน้าโหมดฝึกโหลดชุดข้อสอบได้ และ bundle ล่าสุดไม่มีข้อผิดพลาด

> หมายเหตุ: เอกสารนี้ระบุเฉพาะชื่อ Environment Variables และไม่บันทึกค่าคีย์ลง Git
